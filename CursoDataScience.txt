Teste de Turing ( Alan Turing )
The Imigation Game = Jogo de Imitação
Computing Machine and Intelligence = Artigo de Turing 1950
Filme Ex Machina

 O Teste de Turing funciona da seguinte forma: um interrogador (humano) fará perguntas a duas entidades ocultas; uma delas é um humano e a outra é um computador. A comunicação entre o interrogador e as entidades é feita de modo indireto, pelo teclado, por exemplo. O interrogador tentará, através do “diálogo” realizado entre ele e as entidades, decidir qual dos dois é o humano. O computador será programado para se passar por humano e o humano responderá de forma a confirmar a sua condição. Se, no final do teste, o interrogador não conseguir distinguir quem é o humano, então conclui-se que o computador pode “pensar” segundo o Teste de Turing.

Machine Learning / Aprendizado de Máquina :

É um campo da Ciência da Computação que se concentra em Criar sistemas que são Capazaes de Aprender a partir de um conjunto de dados, sem precisar de Programação Direta

Aplicações :
Reconhecimento de Voz
Análise de Sentimentos
Recomendação
Detecção de Fraudes
Previsões Empresariais

Tipos :
Aprendizado Supervisionado = Treinado com Conjunto de Dados
Aprendizado Não Supervisionado = Treinado sem Nada, apenas com regras
Aprendizado por Reforço = Treinador por série de ações e feedbacks, vão fazer pra ganhar Algoritmo
Aprendizado Profundo = Treinado com camadas ocultas

-----------------------------------------------------
Processo de Aprendizagem em Machine Learning :

Definir uma função de perda ( ou erro ) = Função que mede a diferença entre as previsões feitas pelo modelo e os valores reais para cada exemplo
Otimizar a função de perde : Encontrar valores dos parâmetros que minimizam a função de perda

-----------------------------------------------------
Treinamento em IA :

Processo de aprender os parâmetros do modelo a partir de um conjunto de dados de treinamento ( datasets ).

Por exemplo, se você estiver treinando um modelo para prever se uma pessoa tem diabetes, os dados de treinamento poderiam incluir entradas como idade, peso, pressão arterial e níveis de açúcar no sangue, e a saída seria uma previsão de se a pessoa tem ou não diabetes.

Tipos de Datasets ( Dados de Treinamento )
-Rotulados : Identificação de algo para entender suas características
-Não Rotulados : Identificar padrões / semelhanças nos dados para chegar as conclusões 

-----------------------------------------------------
Validação em IA :
Processo de avaliar como o modelo se sai com novos dados que não foram usados durante o treinamento.

Esse processo ajuda a identificar overfitting ( memorizar dados, mas não é capaz de generalizar para novos casos )
Ex: Fase de jogo
O modelo sai bem na fase 1, porém ele tem dificuldades na fase 2

Validação Cruzada :
Método que envolve particionar aleatoriamente os dados de treinamento em vários subconjuntos e reservar um para avaliação .

Dados de Teste vs Dados de Validação
Testar o modelo após a Conclusão do Treinamento
Validar o modelo durante o Treinamento

-----------------------------------------------------
Teste em IA :
Processo de avaliar como o modelo se sai com um conjunto de dados ainda não visto, como o objetivo de medir a perfomance final.
